Drone Poker Hand Detector Project

Course: None

Date: June 2023

Project Description: The objective of this project was to program a drone to autonomously detect and identify poker hands using computer vision. The drone was designed to fly over a poker table, capture images of the cards, and analyze the images to determine the poker hand of each player.

Specific Role: As the sole developer, I was responsible for the entire project lifecycle, including designing the object detection algorithm, programming the drone's movement and image capture logic, and integrating the computer vision system. My role involved extensive coding in Python and utilizing libraries such as OpenCV and TensorFlow.

Learnings: Through this project, I enhanced my understanding of computer vision and machine learning concepts, particularly in the context of image recognition. I learned how to train and deploy object detection models for card recognition and integrate them with drone hardware. Additionally, I gained practical experience in real-time image processing and autonomous systems programming.

Challenges Overcome: One of the primary challenges was ensuring accurate detection and recognition of poker hands under varying lighting conditions and angles. This required training the model with a diverse dataset and implementing image preprocessing techniques. Another challenge was achieving real-time processing speeds to provide immediate feedback, which involved optimizing both the machine learning model and the drone's control algorithms.

Teamwork/Leadership: While this project was individual, I collaborated with peers working on similar computer vision projects by sharing insights and troubleshooting common issues. I also documented the entire process meticulously, providing a guide for others interested in similar projects and contributing to the collaborative learning environment.

Key Contributions:

  • Poker Hand Detection Algorithm Implementation
  • Real-Time Image Processing
  • Autonomous Movement and Image Capture Logic
  • System Integration and Testing
